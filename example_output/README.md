# Example Output

This folder is intended to store example output files generated by the
analysis pipeline for a reference experiment.  For the paper, the outputs
were produced with the commands listed in the Methods section.

## Expected files

| File | Produced by | Description |
|---|---|---|
| `first_frame.jpg` | `check_video.py` | First frame of the video; used for ROI setup |
| `frame_9.0s.jpg` | `check_video.py` | Reference frame at 9 s |
| `roi_check.jpg` | `set_roi.py` | Reference frame with darkness ROI overlaid |
| `foam_roi_check.jpg` | `set_roi.py` | Reference frame with foam strip ROI overlaid |
| `darkness_features.csv` | `analyze_darkness.py` | Raw per-frame darkness metrics |
| `darkness_features_smooth.csv` | `smooth_features.py` | Smoothed darkness metrics |
| `foam_features.csv` | `analyze_foam.py` | Per-frame foam thickness |
| `combined_plot.png` | `plot_combined.py` | Dual-axis figure for the paper |

## CSV column definitions

### darkness_features.csv / darkness_features_smooth.csv

| Column | Unit | Description |
|---|---|---|
| `time_s` | s | Elapsed time |
| `mean_gray` | — | Mean ROI pixel intensity (0–255) |
| `dark_ratio` | — | Fraction of pixels below threshold |
| `darkness` | — | `1 − mean_gray / 255` |
| `std_gray` | — | Standard deviation of ROI pixel intensities |
| `std_gray_norm` | — | `std_gray / 255` |
| `darkness_smooth` | — | Rolling-mean smoothed darkness *(smooth CSV only)* |
| `dark_ratio_smooth` | — | Rolling-mean smoothed dark_ratio *(smooth CSV only)* |
| `std_gray_smooth` | — | Rolling-mean smoothed std_gray_norm *(smooth CSV only)* |

### foam_features.csv

| Column | Unit | Description |
|---|---|---|
| `time_s` | s | Elapsed time |
| `foam_height_px` | px | Raw foam thickness (0 = no foam detected) |
| `foam_height_smooth` | px | Moving-average smoothed foam thickness |

## Example commands used to produce these files

```bash
cd ../src

python check_video.py \
    --video ../data/InCl3_noZn_0302.mp4 \
    --target_time 9 \
    --output_dir ../example_output

python set_roi.py \
    --image ../example_output/first_frame.jpg \
    --roi 800 775 100 70 \
    --output ../example_output/roi_check.jpg

python set_roi.py \
    --image ../example_output/first_frame.jpg \
    --roi 800 500 125 350 \
    --output ../example_output/foam_roi_check.jpg

python analyze_darkness.py \
    --video ../data/InCl3_noZn_0302.mp4 \
    --roi 800 775 100 70 \
    --threshold 65 \
    --step_time 0.3 \
    --output ../example_output/darkness_features.csv

python smooth_features.py \
    --input ../example_output/darkness_features.csv \
    --window 9 \
    --output ../example_output/darkness_features_smooth.csv

python analyze_foam.py \
    --video ../data/InCl3_noZn_0302.mp4 \
    --roi 800 500 125 350 \
    --output ../example_output/foam_features.csv

python plot_combined.py \
    --foam ../example_output/foam_features.csv \
    --darkness ../example_output/darkness_features.csv \
    --title "InCl3 (no Zn) â 03/02" \
    --output ../example_output/combined_plot.png
```
